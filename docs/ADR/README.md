# Architecture Decision Records (ADR)

Este diretório contém decisões arquiteturais importantes do projeto SGP+.

## Template

Cada ADR deve seguir o formato:

1. **Status**: Proposta | Aceita | Rejeitada | Deprecada
2. **Contexto**: Por que esta decisão foi necessária
3. **Decisão**: O que foi decidido
4. **Consequências**: Impactos positivos e negativos
